# Linux 支撑能力

当前实验环境使用 Ubuntu，本目录只记录服务于 CUDA Kernel 开发与 AI 推理基础设施的 Linux 能力。

## 关注范围

- NVIDIA Driver、CUDA Toolkit、cuDNN 等 GPU 开发环境安装与排查
- `nvidia-smi`、Nsight Systems、Nsight Compute 等 profiling 工具
- 动态库路径、权限、设备文件、容器 GPU 访问
- `perf`、`htop`、`numactl`、`lscpu`、`lsblk` 等系统观察工具
- shell 脚本与自动化实验环境

## 非目标

本目录不作为通用 Linux 入门教程，不系统记录基础命令、发行版使用技巧或桌面环境配置。
